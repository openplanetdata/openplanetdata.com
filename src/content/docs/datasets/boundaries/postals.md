Postal code area boundaries, derived from OpenStreetMap data.

## Overview

| Attribute | Value |
|-----------|-------|
| Update Frequency | Daily |
| Formats | GeoJSON, GeoPackage, GeoParquet |
| Coverage | Global |
| License | Open Database License (ODbL) |

## Access

Files are available via direct download from our Cloudflare R2 storage:

```
https://data.openplanetdata.com/boundaries/postals/
```

## Use Cases

- **Geocoding**: determine which postal area a coordinate belongs to
- **Logistics**: plan delivery zones and coverage areas
- **Analytics**: aggregate data by postal code
