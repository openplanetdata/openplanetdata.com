OpenPlanetData provides administrative boundary data at multiple levels, derived from OpenStreetMap and updated daily. All boundary datasets are available in GeoJSON, GeoPackage, and GeoParquet formats.

<WorldMap />

## Available Boundary Datasets

- **[Continents](/datasets/boundaries/continents/)** — boundaries for all continents and major world regions
- **[Countries](/datasets/boundaries/countries/)** — complete coverage of all recognized countries and territories worldwide
- **[Regions](/datasets/boundaries/regions/)** — administrative regions and subdivisions (ISO 3166-2)
- **[Cities](/datasets/boundaries/cities/)** — city and municipality boundaries
- **[Postals](/datasets/boundaries/postals/)** — postal code area boundaries

## Use Cases

- **Analytics** — aggregate data by geographic regions
- **Geocoding** — determine which administrative area a coordinate belongs to
- **Validation** — verify location data accuracy
- **Visualization** — display boundaries on maps and dashboards
