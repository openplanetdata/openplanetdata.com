Administrative regions and subdivisions (ISO 3166-2) for countries worldwide, derived from OpenStreetMap data.

## Overview

| Attribute | Value |
|-----------|-------|
| Update Frequency | Daily |
| Formats | GeoJSON, GeoPackage, GeoParquet |
| Coverage | Global |
| License | Open Database License (ODbL) |

## Access

Files are available via direct download from our Cloudflare R2 storage:

```
https://data.openplanetdata.com/boundaries/regions/
```

## Use Cases

- **Geocoding**: determine which administrative region a coordinate belongs to
- **Visualization**: display regional boundaries on maps and dashboards
- **Analytics**: aggregate data by administrative subdivision
