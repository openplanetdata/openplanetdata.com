Complete coverage of all recognized countries and territories worldwide, derived from OpenStreetMap data.

## Overview

| Attribute | Value |
|-----------|-------|
| Update Frequency | Daily |
| Formats | GeoJSON, GeoPackage, GeoParquet |
| Coverage | Global |
| License | Open Database License (ODbL) |

## Access

Files are available via direct download from our Cloudflare R2 storage:

```
https://data.openplanetdata.com/boundaries/countries/
```

## Use Cases

- **Geocoding**: determine which country a coordinate belongs to
- **Visualization**: display country boundaries on maps and dashboards
- **Analytics**: aggregate data by country
