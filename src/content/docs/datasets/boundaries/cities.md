City and municipality boundaries worldwide, derived from OpenStreetMap data.

## Overview

| Attribute | Value |
|-----------|-------|
| Update Frequency | Daily |
| Formats | GeoJSON, GeoPackage, GeoParquet |
| Coverage | Global |
| License | Open Database License (ODbL) |

## Access

Files are available via direct download from our Cloudflare R2 storage:

```
https://data.openplanetdata.com/boundaries/cities/
```

## Use Cases

- **Geocoding**: determine which city a coordinate belongs to
- **Visualization**: display city boundaries on maps and dashboards
- **Analytics**: aggregate data by city or municipality
