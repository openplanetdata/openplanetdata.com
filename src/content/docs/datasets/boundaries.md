The Boundaries dataset provides administrative boundary data for continents, countries, and regions in multiple geospatial formats.

## Overview

| Attribute | Value |
|-----------|-------|
| Update Frequency | Daily |
| Formats | GeoJSON, GeoPackage, GeoParquet |
| Coverage | Global |
| License | Open Database License (ODbL) |

## Available Boundaries

### Continents
- Africa
- Antarctica
- Asia
- Europe
- North America
- Oceania
- South America

### Countries
Complete coverage of all recognized countries and territories worldwide.

### Regions
Administrative regions and subdivisions for supported countries.

## File Formats

### GeoJSON
Standard GeoJSON format for maximum compatibility with web applications and GIS tools.

### GeoPackage
SQLite-based format optimized for desktop GIS applications like QGIS.

### GeoParquet
Columnar format optimized for cloud-native analytics and big data workflows.

## Access

Files are available via direct download from our Cloudflare R2 storage:

```
https://data.openplanetdata.com/boundaries/
```

## Data Sources

Boundary data is derived from authoritative open data sources and updated daily to reflect the latest changes.

## Use Cases

- **Geocoding**: Determine which country or region a coordinate belongs to
- **Visualization**: Display boundaries on maps and dashboards
- **Analytics**: Aggregate data by geographic regions
- **Validation**: Verify location data accuracy
